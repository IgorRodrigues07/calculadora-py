# calculadora em python
# 

Uma aplicação robusta de linha de comando (CLI) que centraliza diversas operações matemáticas, desde cálculos básicos de aritmética até funções avançadas de álgebra, sequências lógicas e conversões.

##  Funcionalidades
O projeto conta com um Menu Interativo completo que permite navegar entre as seguintes operações:

* **Operações Básicas:** Soma, Subtração, Multiplicação e Divisão (com tratamento de erro para divisão por zero).
* **Estatística:** Cálculo de Média Aritmética para uma quantidade variável de números.
* **Álgebra:** Resolução de Equações de 2º Grau (Fórmula de Bháskara).
* **Matemática Discreta:** Cálculo de Fatorial e Sequência de Fibonacci.
* **Utilitários:** Gerador de Tabuada dinâmica e Conversão de Unidades.
* **Fluxo de Saída:** Opção dedicada para encerrar o programa com segurança.

##  Diferenciais Técnicos
Este projeto foi desenvolvido com foco em boas práticas de lógica de programação:

* **Modularização:** Código organizado em funções específicas (`def`) para cada operação, facilitando a manutenção.
* **Uso de Bibliotecas:** Integração com a biblioteca padrão `math` para cálculos de precisão (como raízes quadradas).
* **Estruturas de Controlo:** Utilização de laços `while` para manter o programa ativo e blocos `if/elif/else` para a lógica do menu.
* **Experiência do Utilizador (UX):** Mensagens claras no terminal, formatação de resultados e validação de entradas.
